checkCollision determines a collision between two movable entities. It is a general function for all entities. It assumes each entity as rectangular in shape thus each entity has a width and a height. The Algorithm is based on detecting whether the edges (width and height) of the rectangular entities have collided.

Parameters
entity1(one of the entities to be checked for collision)
entity2(One of the entities to be checked for collision)
entity1Width(The width of the 1st rectangular entity)
entity1Height(the height of the 1st rectangular entity)
entity2Width(The width of the 2nd rectangular entity)
entity2Height(The height of the 2nd rectangular entity)
Returns
boolian true if a collision between the entities has occured, false otherwise
